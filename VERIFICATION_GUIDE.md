# Alumni Verification - User Guide

Welcome to the NALUM Alumni Verification System! This guide will help you verify your alumni status and unlock access to all dashboard features.

## Why Verify?

All dashboard features including Profile Management, Alumni Directory, Events, and Messaging are locked until you verify your alumni status. This ensures our platform maintains a trusted community of verified NSUT alumni.

### üé´ Method 1: Verification Code (Fastest)

**When to use:** If you received a 10-digit verification code via email

**Steps:**
1. Click on "Verification Code" card
2. Enter your 10-digit code (format: ABC1234567)
3. Click "Verify Code"
4. ‚úÖ Instant verification!

**Tips:**
- Check your alumni email for the code
- Code can only be used once
- Contact admin if you haven't received a code

### üóÑÔ∏è Method 2: Database Check (Automated)

**When to use:** If your information is in the college alumni database

**Steps:**
1. Click on "Database Check" card
2. Fill in your details:
   - Full Name (as registered with college)
   - Roll Number (optional but helpful)
   - Batch/Graduation Year
   - Branch
3. Click "Check Database"
4. If matches found, select your record
5. Click "Confirm Selection"
6. ‚úÖ Verified!

**What happens if no match?**
- Your request automatically goes to admins for manual review
- No extra steps needed from you
- You'll be notified once reviewed

**Tips:**
- Use your official name (as on college records)
- Double-check batch and branch selection
- Roll number increases match accuracy

---

### üë§ Method 3: Manual Review (Admin Verification)

**When to use:** If database check didn't work or you're not in the system

**Steps:**
1. Click on "Manual Review" card
2. Fill in your details:
   - Full Name
   - Roll Number (if available)
   - Batch/Graduation Year
   - Branch
3. Click "Submit for Review"
4. ‚úÖ Request submitted!
5. Wait for admin approval (2-3 business days)

**What happens next?**
- Admin team reviews your submission
- They verify against college records
- You receive email notification with decision
- If approved, you're verified!

**Tips:**
- Provide accurate information
- Include roll number if you remember it
- Check spam folder for admin notification

### ‚ùå "Verification code is invalid"
- Check if code is exactly 10 characters

- Contact admin if code was already used

### ‚ùå "No matches found"
- Try database check first

- Your request is automatically queued for admin review

### ‚ùå "Request already submitted"

- Check email for updates

### ‚ùå "Fill in all required fields"

## FAQs

**Q: How long does verification take?**
- Code: Instant
- Database: 1-2 minutes
- Manual: 2-3 business days

**Q: Can I use multiple methods?**
- Yes! Try database check first, then manual if needed
- Once verified, no need to verify again

**Q: What if my information has changed?**
- Use the name and details from your graduation records
- Mention any changes in manual review

**Q: Do I need to verify again later?**
- No! Once verified, you're permanently verified
- Your status is saved to your account

**Q: Can I update my information after verification?**
- Yes! Once verified, you can edit your entire profile
- Includes current position, skills, experience, etc.
